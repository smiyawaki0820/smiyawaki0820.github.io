SLUD2020

人工知能学会 言語・音声理解と対話処理研究会 SIG-SLUD 2020 「対話システムライブコンペティション3」のオープントラックに提出し， 我々が構築した対話システム「ILYS aoba bot」が NTT CS研 に次いで参加チーム中２位になりました．

  • Github: ILYS-aoba-chatbot

本コンペティションのオープントラックは「自然，かつ，ユーザが話したい話題について情報交換が可能なシステムであるか」を競うものであるため， システムが話題に即した自然な応答が可能であり，また話題に関する新たな情報を提供できることが可能なシステム構築を目的とし， 我々は話題に即した柔軟な応答が生成可能な系列変換器と，外部知識として Wikipedia の知識ベースを用いることで話題に関する新たな情報を含む応答が生成可能なルールベース応答を統合しました．

Efficient Open-Domain Question Answering

国際会議 NeurIPS 2020 で開催された QA Competition (Efficient Open-Domain Question Answering) に Studio Ousia さんと共同で開発させていただいた「Sōseki」で挑み， Unrestrictedトラックで マイクロソフト社@MSFTResearch，フェイスブック社@facebookai に次いで参加チーム中3位、6GBトラックで参加チーム中2位になりました．

In this competition contestants will develop a question answering system that contains all of the knowledge required to answer open-domain questions. There are no constraints on how the knowledge is stored—it could be in documents, databases, the parameters of a neural network, or any other form. However, three competition tracks encourage systems that store and access this knowledge using the smallest number of bytes, including code, corpora, and model parameters. There will also be an unconstrained track, in which the goal is to achieve the best possible question answering performance with no constraints. The best performing systems from each of the tracks will be put to test in a live competition against trivia experts during the NeurIPS 2020 competition track.

AI王 ~クイズAI日本一決定戦

言語処理学会第27回年次大会（NLP2021） のワークショップの一つとして開催される， クイズを題材とした日本語質問応答システムのコンペティションのオーガナイザの一人として参加させていただいております．

  • NLP2021 ワークショップ2
  • (コンペ詳細) AI王 〜クイズAI日本一決定戦〜
  • リーダーボード
  • Twitter

知識に基づいた質問応答システムの実現は，自然言語処理の大きなゴールの一つです． しかしながら，その実現や評価のためのデータセットの多くは英語に限られています． 本ワークショップでは，クイズを題材とした日本語QAデータセットJAQKETを使って，日本語質問応答システムのコンペティションを開催します． 具体的には，事前にシステムを構築して持ち寄っていただき，その場で勝者を決定するライブコンペを行います． また，各チームによるシステム報告を行っていただきます． 機械読解や質問応答の第一人者による最新の研究動向の解説や，クイズ作家を招いてご講演いただく企画も予定しています．
